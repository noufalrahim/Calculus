# CALCULUS
## Calculus is an open-source accounting software developed to enhance the day-to-day sales and purchases of entrepreneurs.

- [Usage](#Usage)

## Installation
   ```sh
   git clone https://github.com/noufalrahim/Calculus.git
   npm install
   cd app
   npm install
```
## Usage
  ```sh
  npm start
  cd app
  npm run dev
  ```
